# Ubuntu + Node.js + PM2 + GitHub Actions 自动部署教程

适用于：

- Ubuntu 服务器
- Node.js 项目
- PM2 管理进程
- GitHub 仓库
- 自动 git pull + npm install + pm2 restart

---

# 一、服务器准备

## 1. 安装 Git

```bash
apt update
apt install git -y
````

验证：

```bash
git --version
```

---

# 二、安装 Node.js（NVM 推荐）

安装 NVM：

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
```

加载：

```bash
source ~/.bashrc
```

安装 Node：

```bash
nvm install 22
```

查看版本：

```bash
node -v
npm -v
```

---

# 三、安装 PM2

安装：

```bash
npm install -g pm2
```

查看：

```bash
pm2 -v
```

---

# 四、拉取项目

创建目录：

```bash
mkdir -p /root/product
cd /root/product
```

克隆项目：

```bash
git clone 仓库地址
```

进入项目：

```bash
cd finance-app
```

切换 dev 分支：

```bash
git checkout dev
```

---

# 五、手动启动项目

安装依赖：

```bash
npm install
```

启动：

```bash
pm2 start ./start.sh --interpreter bash --name stock
```

查看：

```bash
pm2 list
```

日志：

```bash
pm2 logs
```

---

# 六、PM2 开机自启

保存进程：

```bash
pm2 save
```

生成开机启动：

```bash
pm2 startup
```

它会输出一条命令。

复制再执行一次。

---

# 七、生成 SSH Key

生成：

```bash
ssh-keygen -t ed25519 -N ""
```

查看公钥：

```bash
cat ~/.ssh/id_ed25519.pub
```

查看私钥：

```bash
cat ~/.ssh/id_ed25519
```

---

# 八、配置 GitHub Deploy Keys

GitHub：

```text
仓库
→ Settings
→ Deploy keys
→ Add deploy key
```

填入：

```text
id_ed25519.pub
```

公钥内容。

勾选：

```text
Allow write access
```

---

# 九、配置 authorized_keys

服务器执行：

```bash
mkdir -p ~/.ssh
chmod 700 ~/.ssh
```

编辑：

```bash
nano ~/.ssh/authorized_keys
```

粘贴：

```bash
cat ~/.ssh/id_ed25519.pub
```

输出内容。

保存。

权限：

```bash
chmod 600 ~/.ssh/authorized_keys
```

---

# 十、GitHub Secrets

GitHub：

```text
Settings
→ Secrets and variables
→ Actions
```

添加：

---

## HOST

服务器 IP

例如：

```text
1.2.3.4
```

---

## SSH_KEY

填：

```bash
cat ~/.ssh/id_ed25519
```

私钥完整内容。

必须包含：

```text
-----BEGIN OPENSSH PRIVATE KEY-----
-----END OPENSSH PRIVATE KEY-----
```

---

# 十一、创建 GitHub Actions

创建：

```text
.github/workflows/deploy.yml
```

内容：

```yaml
name: Deploy

on:
  push:
    branches:
      - dev

jobs:
  deploy:
    runs-on: ubuntu-latest

    steps:
      - name: Deploy
        uses: appleboy/ssh-action@v1.0.3

        with:
          host: ${{ secrets.HOST }}
          username: root
          key: ${{ secrets.SSH_KEY }}

          script: |
            export PATH=$PATH:/root/.nvm/versions/node/v22.22.2/bin

            cd /root/product/finance-app

            git fetch origin
            git checkout dev
            git pull origin dev

            npm install

            pm2 restart stock || \
            pm2 start ./start.sh --interpreter bash --name stock
```

---

# 十二、触发自动部署

本地：

```bash
git add .
git commit -m "deploy"
git push origin dev
```

GitHub Actions 会自动：

* SSH 登录服务器
* git pull
* npm install
* pm2 restart

# 十三、常见问题

---

## 1. npm: command not found

说明：

```text
PATH 没加载
```

解决：

```yaml
export PATH=$PATH:/root/.nvm/versions/node/v22.22.2/bin
```

---

## 2. ssh.ParsePrivateKey

说明：

```text
SSH_KEY 填错
```

必须填：

```text
id_ed25519
```

不是：

```text
id_ed25519.pub
```

---

## 3. unable to authenticate

说明：

```text
authorized_keys 没配置
```

---

## 4. pm2 重复启动

不要：

```bash
pm2 start
```

推荐：

```bash
pm2 restart stock || pm2 start
```
